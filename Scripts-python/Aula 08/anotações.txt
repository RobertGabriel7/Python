Utilizando Módulos:

Exemplo: quando eu compro um carro básico, ele vem com bãsico para funcionar, porém, eu posso colocar um som melhor do que veio. Ou seja, eu posso adicionar módulos nele, melhorando tudo do carro.

O python de forma semelhante, vem com o básico pré instalado, mas eu posso adcionar mais módulos nele. Por isso que o python é um linguagem rápida, pelo fato de ter apenas os recursos necessarios.

>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

Agora, se eu precisar de recursos adicionais, eu posso importar (import). No exemplo abaixo vai importar tudo refêrente a banco de carro.

*For example with car seat (banco/assento de carro):

import seat 

Assim ele importa todas as portas de carros para o programa

*Another example: 
Caso eu queira importar somente um livro de uma biblioteca inteira:

from biblioteca import as crônicas de nárnias
